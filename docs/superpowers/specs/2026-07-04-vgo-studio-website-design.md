# VGO 工作室官网设计文档

## 1. 概述

为 VGO（原 VGOApp）软件工作室设计并开发一个单页滚动官网，展示三款产品：

- [MDDock](https://mddock.com/) —— local-first Markdown 办公套件 + AI 知识库
- [MacSlim](https://github.com/edwin-hao-ai/MacSlim) —— 面向开发者的 Mac 清理工具
- [MouseClaw](https://github.com/edwin-hao-ai/MouseClaw) —— 像素风桌面 AI 宠物

整体风格：Apple 精致玻璃风（glassmorphism），深色主调，强调高级感、产品质感和微动效。

## 2. 品牌与命名

- **工作室名称**：VGO
- **Slogan 候选**：
  - 中文：「做你真正会用的工具」
  - 英文："Tools you actually use."
- **品牌调性**：克制、精致、本地优先、开发者友好、AI 原生。

## 3. 视觉设计系统

### 3.1 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景主色 | `#0a0a0f` | 接近纯黑，带轻微蓝调 |
| 背景次级 | `#12121a` | 卡片/区块底层 |
| 玻璃高光 | `rgba(255,255,255,0.08)` | 玻璃卡片边框/高光 |
| 玻璃填充 | `rgba(255,255,255,0.04)` | 玻璃卡片背景 |
| 主强调色 | `#60a5fa` | 冰蓝，用于按钮、链接、高亮 |
| 辅强调色 | `#a78bfa` | 淡紫，用于渐变和悬浮态 |
| 文字主色 | `#f8fafc` | 近白 |
| 文字次色 | `#94a3b8` | 灰蓝，用于描述文字 |

渐变方案：
- 主渐变：`linear-gradient(135deg, #60a5fa 0%, #a78bfa 100%)`
- 背景光晕：大半径模糊径向渐变，随鼠标微动（`#60a5fa` / `#a78bfa`，透明度 0.15）

### 3.2 字体

- 中文：`PingFang SC`, `Microsoft YaHei`, sans-serif（系统字体，轻量）
- 英文/数字：`Inter`, `SF Pro Display`, system-ui
- 标题字重：700–800
- 正文字重：400–500
- 行高：1.6（正文），1.1（大标题）

### 3.3 玻璃效果规范

- `backdrop-filter: blur(20px)`
- `background: rgba(255, 255, 255, 0.04)`
- `border: 1px solid rgba(255, 255, 255, 0.08)`
- `border-radius: 24px`（大卡片）/ `16px`（小卡片）
- 阴影：`0 8px 32px rgba(0, 0, 0, 0.3)`

## 4. Logo 设计

### 4.1 方向

采用**抽象玻璃棱镜/晶体图标**，不用字母组合，以便未来承载更多 App。

### 4.2 概念描述

- 一个由多个透明三角面组成的几何晶体，微微倾斜，呈现折射效果。
- 每个面用渐变填充（蓝到紫），带有半透明高光和细边框。
- 整体轮廓简洁，小尺寸下仍清晰可辨。
- 寓意：VGO 的产品像棱镜一样，把复杂工作折射成简单、可用的工具。

### 4.3 变体

- **主 Logo**：彩色玻璃晶体 + "VGO" 文字
- **图标版**：仅彩色晶体，用于 favicon、社交媒体
- **单色版**：纯白/纯黑晶体，用于水印、打印

### 4.4 实现方式

- 优先使用 SVG 绘制，确保无损缩放。
- 可导出 32×32、64×64、180×180、512×512 PNG 作为 favicon/PWA 图标。

## 5. 页面结构与内容

单页滚动，共分为 7 个区块：

### 5.1 Navigation（固定顶部玻璃导航）

- 左侧：VGO Logo + 文字
- 中部：锚点链接（产品、关于、联系）
- 右侧：GitHub 图标链接
- 滚动后导航栏背景变实、添加底部分割线

### 5.2 Hero

- 大标题："VGO"
- 副标题："做你真正会用的工具"
- 简短说明："本地优先、AI 原生、为开发者和效率爱好者打造。"
- CTA 按钮："探索产品"（滚动到 Products）
- 背景：动态渐变光晕 + 细微粒子/网格线（可选）
- Logo 晶体以 3D 轻微旋转/悬浮动效出现

### 5.3 Products（三款产品展示）

每款产品一个玻璃卡片，横向排列（桌面端），垂直堆叠（移动端）。

卡片内容：
- 产品图标/示意图
- 产品名称
- 一句话定位
- 3 个核心亮点（带图标）
- "了解更多" 按钮（外链到各自官网/GitHub）

#### MDDock

- 定位："本地优先的 Markdown 办公套件"
- 亮点：
  - Word/Excel/PPT 与 Markdown 双向转换
  - AI 知识库，文件永远在本机
  - 免费浏览器工具 + 桌面端深度体验

#### MacSlim

- 定位："开发者信任的 Mac 清理工具"
- 亮点：
  - Rust + Tauri，安装包仅 ~12 MB
  - 支持 npm、Docker、Xcode、Homebrew 等开发者缓存
  - 三层安全保护，零上传隐私

#### MouseClaw

- 定位："像素风桌面 AI 宠物"
- 亮点：
  - 按住说话即可调用 Claude / Codex / 通义千问等 14+ AI
  - 100% 本地语音识别，零数据上传
  - 9 款皮肤、长期记忆、定时任务

### 5.4 Why VGO（工作室理念）

三个玻璃小卡片：

1. **本地优先** —— 你的数据默认留在设备上，不上传、不锁定。
2. **AI 原生** —— 不是后期加功能，而是从设计之初就为 AI 工作流打造。
3. **开发者品味** —— 追求性能、隐私和细节，拒绝臃肿 Electron。

## 6. 交互动效

### 6.1 全局

- 鼠标跟随光晕：背景有一层轻微移动的径向渐变，随鼠标位置偏移。
- 滚动揭示：各区块进入视口时 fade-up + translateY。
- 平滑滚动：锚点跳转使用 `scroll-behavior: smooth`。

### 6.2 玻璃卡片

- 悬浮：轻微上移（`translateY(-8px)`）、边框亮度增加、阴影加深。
- 倾斜/光泽（可选）：鼠标在卡片上移动时产生 3D 倾斜和高光扫过效果。

### 6.3 Hero

- Logo 晶体缓慢上下悬浮（`translateY` 正弦波动）。
- 标题文字逐字或逐行淡入。
- CTA 按钮 hover 渐变流动。

### 6.4 性能要求

- 使用 `will-change` 和 `transform`/`opacity` 做动画，避免重排。
- 粒子/复杂效果在移动端降级或关闭。
- 支持 `prefers-reduced-motion` 媒体查询。

## 7. 技术栈

- **框架**：Vite + React + TypeScript
- **样式**：Tailwind CSS v4
- **动效**：Framer Motion（滚动揭示、悬浮、页面转场）
- **图标**：Lucide React
- **构建产物**：静态 HTML/CSS/JS，可部署到任意 CDN / GitHub Pages / Vercel
- **包管理器**：npm / pnpm

## 8. 响应式断点

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 移动端 | < 640px | 单列、紧凑导航（汉堡菜单）、卡片垂直堆叠 |
| 平板 | 640px–1024px | 两列或单列、适度间距 |
| 桌面 | > 1024px | 三列产品卡片、大间距、完整动效 |
